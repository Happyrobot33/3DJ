You cannot distribute this avatar or custom variants in other servers or games.


------------------------------------------------------------------------------------------------------------------------------------------
You can create public Kita'vali avatars. 
------------------------------------------------------------------------------------------------------------------------------------------
It is appreciated if you leave the credit gear with it's texture / Kita'vali discord link in the head of the avatar for public creations 
or alternatively place the credit gear somewhere visible in your avatar world.

No Kita'vali avatars promoting hate are permitted.



------------------------------------------------------------------------------------------------------------------------------------------
You can use this avatar as a vtuber avatar base.
------------------------------------------------------------------------------------------------------------------------------------------
You can use the Kita'vali avatar for streaming content/vtuber stuff as long as you credit @岵坚布  (Rai Kitamatsu) for the base.

You cannot use/distribute the avatars base files in merch. 
You can use the avatar in pictures and renders of your own kita'vali vtuber character in merch but it needs to be as <Whatever your name is here> merch. Not Kita'vali merch.



------------------------------------------------------------------------------------------------------------------------------------------
You are not allowed to sell the Kita'vali or any of it's base assets.
------------------------------------------------------------------------------------------------------------------------------------------
You are not allowed to sell the avatar or any of it's base files.

You can sell modifications, clothing and basically any other content you've created for this avatar. 
This content shall not include a copy of the avatar with it.

You can do commissions for this avatar.

You are not allowed to charge for the avatar itself in commissions. Only for the work you've done.

The default textures can be used in paid commissions or products as long as they're altered and not just the default textures. 
You are not allowed to sell the basic default textures for the avatar as they are.

The shading can be used in any paid products or commissions.
For example, Clothing, Skin commissions, etc.

------------------------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------------------------
Credits/Server link 
------------------------------------------------------------------------------------------------------------------------------------------
The default assets have a server link included on a gear in their head. It is requested that you leave this in for public avatars so people know where to find the base assets from. :BunnyHeart: 

If you use these assets in an animation, renders or other creations please include "Kita'vali base created by @RaiKitamatsu" somewhere in the description.



------------------------------------------------------------------------------------------------------------------------------------------
Derivative works
------------------------------------------------------------------------------------------------------------------------------------------
Derivative works need to be credited as a Kita'vali derivative and cannot be sold or distributed as new avatars.
Avatars heavily based on the appearance of the avatar but without using the base mesh are still derivatives. 
Basically the 3D equivalent of tracing an avatar. Don't do this. I've seen it happen a few times recently and it's pretty shitty to do. Doing this will get you named and banned.
